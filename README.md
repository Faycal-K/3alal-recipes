# 3alal-Recipes
This is a project given by TOP (The Odin Project), which includes using the following technologies:
- Hypertext Markup Language (HTML);
- GIT(Hub);
- GNOME Terminal within Xubuntu;
- Visual Studio Code (VS code).

## Goal
The goal is to create a website which contains my favorite recipes.

## Skills
These are the skills needed to complete this website:
- Use the GNOME Terminal to create, delete, rename and move files and folders;
- Edit the .HTML files through VS code;
- Commit every atomic change using Git. If needed revert change(s);
- Push the changes to Git(hub).
